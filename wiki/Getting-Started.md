# Getting Started

Let's get you sharing files with QuickDrop in under 5 minutes! 🚀

## 📱 Before You Start

QuickDrop works on **any device with a modern web browser**:
- ✅ Chrome/Chromium (recommended)
- ✅ Firefox
- ✅ Safari
- ✅ Edge
- ✅ Mobile browsers (iOS Safari, Chrome Mobile)

**All you need: a stable internet connection.**

---

## 🎯 Sending a File

### Step 1: Open QuickDrop

Go to: **[quickdrop-share.adheesharavindu001.workers.dev](https://quickdrop-share.adheesharavindu001.workers.dev)**

### Step 2: Choose "Send"

Click the **"Send"** button on the home screen.

### Step 3: Select Your File

- Click the file upload area, or
- Drag and drop a file directly

You can select any file type—documents, images, videos, etc. **No size limit.**

### Step 4: Share the QR Code

Once the file is ready, you'll see:
- A **QR Code** (scan this with the receiver's camera app)
- A **Room ID** (copy and share manually if preferred)

### Step 5: Wait for Connection

Once the receiver joins:
- The transfer starts **automatically**
- Watch the progress bar
- Transfer completes in seconds to minutes depending on file size

✅ **Done! File sent securely.**

## 📥 Receiving a File

### Step 1: Open QuickDrop

Go to: **[quickdrop-share.adheesharavindu001.workers.dev](https://quickdrop-share.adheesharavindu001.workers.dev)**

### Step 2: Choose "Receive"

Click the **"Receive"** button on the home screen.

### Step 3: Join the Room

**Option A: Scan QR Code**
- Open your phone's camera app
- Point it at the sender's QR code
- Tap the notification to open QuickDrop

**Option B: Enter Room ID**
- Copy the room ID from the sender
- Paste it into the text field
- Click "Join"

### Step 4: Wait for Transfer

Once connected:
- The sender's file appears on your screen
- Progress bar shows transfer status
- Click the **"Download"** button when complete

✅ **File downloaded to your device!**

## 💬 Sending Text

QuickDrop also lets you share text snippets instantly.

### Steps:

1. Click **"Send"** → **"Text"**
2. Type or paste your text
3. Share the QR code or room ID
4. Receiver clicks **"Receive"** → scans/joins
5. Receiver clicks **"Copy"** to copy the text

---

## 🔄 Start Another Transfer

After each transfer completes, you can:
- Click **"Start another transfer"** to go again
- Stay in the same session (no page reload needed)
- New transfer, new room ID automatically

---

## 📱 Mobile + Desktop

QuickDrop works great for **mixed scenarios**:

- **Phone to Phone**: Scan QR with device 1, receive on device 2
- **Desktop to Phone**: Share file from computer, receive on mobile
- **Phone to Desktop**: Or vice versa

**Pro Tip:** Use a browser on Mobile instead of deep-linking for best reliability.

---

## ⚡ Pro Tips

- **Large Files?** Patience pays off. WebRTC handles it, just takes longer.
- **Slow Connection?** Try again. If STUN fails, TURN attempts a backup path.
- **Close the Browser?** Transfer fails. Keep the tab open until done.
- **Multiple Files?** Transfer one at a time. Start another transfer after each completes.
- **Want Privacy?** Share room IDs only over encrypted channels (Signal, WhatsApp, etc.).

## ❓ Common Questions

**Q: Is my file safe?**  
A: Absolutely. Your file goes peer-to-peer directly from sender to receiver. No server storage.

**Q: How long does a transfer take?**  
A: Depends on file size and connection speed. Typically seconds for small files, minutes for large ones.

**Q: What if the connection drops?**  
A: The transfer stops. Start a new transfer to retry.

**Q: Can I pause and resume?**  
A: Not yet. We're working on it! For now, use resume-capable tools for large files.
